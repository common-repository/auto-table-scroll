== Description ==

Adds horizontal scroll to big tables in content if they don't fit, useful on mobile viewport.


== Installation ==

After you've downloaded and extracted the files:

1. Upload the complete 'auto-scroll-tables' folder to the '/wp-content/plugins/' directory OR install via the plugin installer
2. Activate the plugin through the 'Plugins' menu in WordPress
3. And yer done!
